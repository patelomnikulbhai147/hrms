HRMS Mobile API — Base URLs
=================================

Mobile auth base path:  /api/v1/auth

Development (local backend):
  http://localhost:5000

Production (through the nginx same-origin proxy on the frontend server):
  http://<YOUR_SERVER_IP_OR_DOMAIN>
  (the app calls http://<server>/api/v1/auth/... — nginx forwards /api to the backend)

Full example endpoint:
  http://localhost:5000/api/v1/auth/login

Set ONE base URL in your Flutter app config and switch it per environment.
